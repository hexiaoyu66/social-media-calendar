## 📖 项目背景 (Background)

在当今内容创作高频迭代的环境下，创作者常面临“空白页恐惧”及“内容结构失衡”的痛点。

本项目旨在通过 Vibe Coding 的开发模式，构建一个极简、直观且具备专业营销逻辑的工具。它能够将复杂的账号定位快速拆解为可执行的内容计划，帮助创作者在保持高产的同时，确保内容覆盖价值分享、情感链接与流量借势。

## 🛠️ 交付物一：可运行的内容日历工具

- 在线访问：[点击此处在线运行](https://hexiaoyu66.github.io/social-media-calendar/)
- 本地运行：下载本仓库的 `index.html` 并在浏览器打开

### 技术亮点

- **轻科技 UI**：采用全宽响应式网格布局与玻璃拟态（Glassmorphism）视觉设计
- **动态逻辑**：根据用户输入的“账号定位”实时生成定制化建议
- **交互增强**：支持执行难度（低 / 中 / 高）实时筛选，优化创作路径

## 📖 使用说明 (User Guide)

1. **输入配置**：在顶部左侧输入框描述您的账号定位（如：领域、受众、风格），右侧小猫形象将作为您的 AI 助手实时陪伴。
2. **选择周期**：点击切换“1周”或“1个月”计划。
3. **一键生成**：点击“生成日历”按钮，工具将按内容矩阵逻辑填充下方日历。
4. **筛选过滤**：点击右侧难度标签，可快速高亮特定执行难度的内容，便于排期。

## 🧠 交付物三：内容策略说明 (核心逻辑)

本工具的生成引擎基于 **5-3-2 社交媒体增长模型** 开发：

- **价值内容 (50%)**：重点在于提供行业洞察与干货。目的是建立“专家”人设，吸引精准粉丝。
- **情感 / 互动 (30%)**：通过幕后故事或提问，打破冷冰冰的屏幕隔阂，提升账号权重。
- **热点 / 借势 (20%)**：结合节假日和行业大事件进行流量收割，完成最终的品牌或商业转化。
